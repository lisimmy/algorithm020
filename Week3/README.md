递归：
	数学归纳法： 找规律，向后证明
	递归的4个步骤
	
	自相似， 重复性、 重复性的子问题， 可以使用递归解决
	
	拒绝人脑递归的情况

分治：
	分治和回溯是特殊的递归
	
	1. 分治， 将问题，转换为重复性的子问题
	
	2. 组合子问题的结果
	
	分治的问题例子：小写字符串-> 大写字符串
	

分治和泛型递归的区别：

	分治，最后需要将结果进行组合
	递归： 最后可能不需要进行结果的组合
	
一般的递归的模板：（泛型递归）

	1. 递归结束条件
	2. 递归逻辑处理--处理当前层的逻辑
	3. 处理子问题，自调用
	4. 处理状态


分治代码模板

	1. 递归结束条件
	2. 递归逻辑处理--处理当前层的逻辑
	3. 下探一层，处理子问题，自调用
	4. 组合结果，将结果组合在一起
	5. 处理状态
	
回溯：
	回溯法通常用最简单的遡归方法来实现，在反复重复上述的步骤后可能出现两种情况
	
	① 找到一个可能存在的正确的答案
	
	② 在尝试了所有可能的分步方法后宣告该问题没有答案。
